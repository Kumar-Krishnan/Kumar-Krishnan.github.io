Projects

Snap Shot

An app for taking psychological tests aimed at measuring depression, anxiety, and other properties. The app allows users to take snapshots of their mental health across multiple dimensions and then view the trend of their scores across time in graphical format.

Technologies Utilized

Front-End: React.js
Back-End: Ruby on Rails
Data Base: PostgreSQL
Misc: Plot.ly(graphing), Semantic-UI-React(styling)

Velocity

An app that allows users, whether they be individuals or organiations to pick and set ten quotes from a set of 1500 that are randomly displayed. The ten values help define the values that they are currently moving towards currently. The values are stored in a values board that allows for only ten values max. The addition of a new value when the value board is full pushes the tenth value out of the board.

Technologies Utilized

Front-End: React.js
Back-End: Node.js
Data Base: MongoDB
Misc: API(to paperquotes)

Connect Four

This is a single page website that allows two users to play Connect Four against each other. It alternates the lead player each round, it checks for all win conditions and the sole draw condition and stores the win count of both players. The theme of the movie TRON inspired the visual design of this app.

Technologies Utilized

HTML, CSS, Javascript, Jquery.

Community Task-Force

Community Task-Force is an app that allows users to post current problems or issues facing their communities, create task-forces to deal with those issues, and sign up as individuals to volunteer for each task force.

Technologies Utilized

Front-End: HandleBars.js, HTML
Back-End: Node.js
Data Base: MongoDB
